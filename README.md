# MineAI

AI-assisted mining game built for the Celo Colombia Hackathon.

## Overview

MineAI is a mobile-first idle mining game where players can mine resources manually or activate AI Managers that automate mining decisions.

The project explores how transparent AI agents can assist players while providing explainable decision-making through detailed decision logs and confidence scoring.

## Core Features

### Manual Mining

* Mine Gold
* Mine Emeralds
* Mine Diamonds
* Energy-based resource extraction

### AI Manager System

* Free Trial Manager
* Basic AI Manager
* Premium AI Manager
* Automated mining decisions

### Decision Transparency

* Decision logs
* Mining rationale
* Confidence scores
* Resource tracking

### Resource System

* Gold
* Emeralds
* Diamonds
* Energy regeneration

## MiniPay Integration

MineAI uses MiniPay as the unlock mechanism for premium AI Managers.

Current MVP includes:

* MiniPay onboarding flow
* Manager unlock system
* Premium automation path
